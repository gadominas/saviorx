In game keyboard shortcuts

In map preview

  • Create Asteroid - LeftShit + Mouse Click
  • Create roadster - LeftShit + Mouse Click + Q
  • Create astronau - LeftShit + Mouse Click + A
  • Delete all spwanable artifacts - Key Delete
  • Delete last spwannable artifact - Key Backspace
  • Drag a planet or spawnable artifact - Mouse Click & Drag
  • Zoom In - LeftShit + UpArrow
  • Zoom Out - LeftShit + DownArrow
  • Move camera - LeftCommand + Mouse Move

Mission board

  • Start - Key Return (Enter). It activates only when leaf unit is deployed
  • Disable/enable a unit - LeftShit + Mouse Click =
  • Lock the unit - LeftShit + Mouse Click (on a selected deployed unit)

In play mode

  • Exit play mode- Key ESC (escape)

Development tools

  • Show/Hide a debug console - LeftControl + Key C
  • Clear user settings - LeftControl + Key R
