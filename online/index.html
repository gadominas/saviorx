Install guide on Android

APK can be downloaded from: SaviorX.com site

  • 1. Open URL. Most likely you will get a warning that this file may harm your device. Click "OK".
  • 2. File will start to download. When download complete, click on notification about complete download.
  • 3. A dialog will appear asking whether to install application. Click "INSTALL". (if this dialog fails to appear, find out "saviorx.apk" file in downloads folder and double click to execute it)
  • 4. Confirmation screen will appear listing application rights. Click "INSTALL"
  • 5. Another dialog may appear asking for install location. Select "Internal device storage", click "OK"
  • 6. When instalation finished, find out application in drawer and run.
